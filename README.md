## 제목
반려동물 케어 시스템

## 사용언어 및 개발환경
- 사용언어 : C, HTML, CSS, JavaScript, PHP, SQL
- 개발환경
  - Vim, Visual Studio Code
  - Apache Web Server
  - MySQL Database Server
  - Raspberry Pi B+
  - Rasbian Linux

## 주요 기능
- 자동 수위 조절 시스템
- 자동 배급 시스템
- 원격 모니터링 시스템
- 환경 파악 시스템

## 사용한 센서
무게센서, 카메라, 온습도센서, 워터펌프, Fan, 부저, Led Bar
